# Security Policy

## Supported Versions

Only the latest `main` branch version and the latest published Chrome Web Store version receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it privately via email to `kanjseok@gmail.com` (or open a GitHub Security Advisory if enabled). Please do **NOT** create a public issue for security vulnerabilities.

We expect to acknowledge your report within 48 hours and typically plan to release a fix within 90 days.

## Disclosure Policy

We ask that you do not publish information about the vulnerability until we have had an opportunity to address it and release an update.

## Security Update Process

Once a vulnerability is resolved, a patch will be released and communicated via GitHub Releases.
